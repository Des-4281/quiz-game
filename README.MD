# JS Quiz

A JavaScript quiz app built with only HTML, CSS, and JavaScript. Focuses on DOM manipulation, event listening, and arrays/objects to function. 

## Features

- Questions load one at a time
- Immediate visual feedback on correct and incorrect answers
- Score tracked throughout and displayed at the end with a percentage
- Restart functionality resets the quiz from the beginning

## What I Learned

- DOM manipulation - selecting elements and updating content dynamically
- Event listeners - responding to user clicks
- Arrays and objects - storing and accessing structured quiz data
- Conditional logic - checking answers and tracking state
- CSS specificity - managing styles that change based on JavaScript-added classes

## How to Run

1. Clone the repo
2. Open `index.html` in your browser
3. No install or build step needed
